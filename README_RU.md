## ✨ Возможности

**🎯 Управление задачами**
- Категории с цветами • Иерархия подзадач • Уровни приоритета • Система тегов • Быстрые действия

**📊 Продвинутая статистика**
- Тепловая карта 365 дней • Серии выполнения • Недельные/часовые графики • Аналитика продуктивности

**🔔 Умные уведомления**
- Напоминания с учётом часовых поясов • Функция отложить • Быстрые действия • Фоновая обработка

**💾 Резервное копирование**
- Автобэкапы (день/неделя/месяц) • GZip сжатие • Простое восстановление • Экспорт данных

**🎨 Красивый дизайн**
- Material You темы • AMOLED тема • Светлая/Тёмная/Системная • Полноэкранный интерфейс

**🌍 Локализация**
- 15 языков: English, Русский, 中文, العربية, فارسی, Español, Français, Deutsch, Italiano, Türkçe, Tiếng Việt, 한국어, Português, Polski

## 🛠️ Сборка из исходников

### Требования
- Flutter SDK 3.41.0 или выше
- Dart SDK 3.11.0 или выше

### Шаги
```bash
# Клонируйте репозиторий
git clone https://github.com/darkmoonight/Planly.ai.git
cd Planly.ai

# Установите зависимости
flutter pub get

# Сгенерируйте код (для базы данных Isar)
dart run build_runner build

# Запустите приложение
flutter run

# Соберите для продакшена
flutter build apk --release  # Android
```
