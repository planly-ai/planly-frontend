# ⚙️ 调试配置快速指南

## 📍 配置文件位置

```
lib/app/constants/debug_config.dart
```

## 🎛️ 配置项说明

### 示例数据配置

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `createSampleChatSession` | `true` | 启动时自动创建示例聊天会话（包含所有类型卡片） |
| `autoRemoveOldSampleSessions` | `false` | 创建新示例数据前自动删除旧的示例数据 |

## 🚀 快速切换

### 开发环境配置

```dart
class DebugConfig {
  static const bool createSampleChatSession = true;
  static const bool autoRemoveOldSampleSessions = false;
}
```

### 生产环境配置

```dart
class DebugConfig {
  static const bool createSampleChatSession = false;
  static const bool autoRemoveOldSampleSessions = false;
}
```

## 📝 使用示例

### 启用示例聊天会话

```dart
// 在 debug_config.dart 中设置：
static const bool createSampleChatSession = true;
```

然后在 `main.dart` 的 `initializeApp()` 中会自动调用：

```dart
if (DebugConfig.createSampleChatSession) {
  if (DebugConfig.autoRemoveOldSampleSessions) {
    await removeSampleChatSessions();
  }
  await createSampleChatSessionWithAllCards();
}
```

## 🎯 最佳实践

1. **使用 const 常量** - 编译时优化，无运行时开销
2. **集中管理** - 所有调试开关集中在一个文件中
3. **清晰命名** - 配置项名称清晰表达功能
4. **环境分离** - 开发和生产环境配置明确区分

## 📂 相关文件

- 配置文件：`lib/app/constants/debug_config.dart`
- 示例数据：`lib/app/data/sample_chat_session.dart`
- 主入口：`lib/main.dart`
